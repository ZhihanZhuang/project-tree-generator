# TreeForge AI 

Preview an AI-generated project tree, generate starter code into your workspace, export to git, and iterate via chat-style requests.

## Commands

- **TreeForge AI: Open** (`treeforge.open`)

## Development

1. Install dependencies

```bash
npm install
```

2. Build

```bash
npm run compile
```

3. Run the extension

- Open this folder in VS Code
- Press `F5` (Run Extension)
- Run the command **TreeForge AI: Open**
